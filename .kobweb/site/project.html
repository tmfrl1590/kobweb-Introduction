프로젝트

온누리 상품권 기반의 전통시장 이커머스 플랫폼 온누리오더 온더

(주)더드림메이커스2024.01 ~ 2024.05
프로젝트 설명 : 전통시장과 상점가에서 사용되는 온누리 상품권 기반의 이커머스 플랫폼(온더)으로, 상용화를 목표로 개발된 프로젝트
프로젝트 배경 : 온더는 R&D 연구개발 과정에서 개발되었으나, 쿠폰 도메인 문제와 복잡한 코드 구조 등으로 인해 상용화에 어려움이 있었고, 이를 해결하기 위해 재개발이 결정됨
문제
· 복잡한 코드 구조 : 단일 액티비티에 비즈니스 로직이 집중되어 코드 복잡도가 높음
· 유지보수 어려움 : 서버 응답 코드와 컬럼명이 하드코딩되어 있고, 개발 문서가 없어 유지보수에 제약
· 쿠폰 발급 제한 : 쿠폰 발급이 매장에 한정되어, 통합관리자가 유연하게 운영할 수 없는 구조
· API 중단 : 제로페이 기반 모바일 온누리 상품권 API가 갑자기 중단되어 결제 기능이 중단됨
해결
· Java 기반 프로젝트를 Kotlin 및 Jetpack Compose로 100% 마이그레이션하여 코드 간결성과 유지보수성 향상· MVVM 패턴과 멀티모듈 아키텍처를 도입하여 프로젝트 구조 모듈화 및 유지보수성 개선· 하드코딩된 서버 응답 코드를 Enum 클래스로 리팩토링하여 코드 가독성과 유지보수성 개선· 테스트 코드를 도입하여 앱의 안정성을 높이고, 배포 시 발생할 수 있는 문제를 최소화· 쿠폰 발급 주체를 통합관리자와 매장 모두 발급 가능하도록 재설계· 스마트로 PG 연동을 통한 카드형 온누리 상품권 결제 시스템 구현· Coroutine 기반의 비동기 네트워크 통신으로 결제 API 요청 속도를 개선하고, 결제 실패 시 오류 처리 로직을 추가하여 결제 안정성 향상

테이블오더 온테이블

(주)더드림메이커스2024.06 ~ 2024.08
프로젝트 설명 : 테이블 오더 시스템 (온테이블)
프로젝트 배경 : 온더 서비스만으로는 경쟁력이 부족하다는 판단에, 매장에서 쉽게 사용할 수 있는 테이블 오더 서비스를 추가로 제공하기 위해 개발을 결정하게 됨
· 약 5개 매장, 30대의 태블릿에서 운영되는 테이블 오더 시스템 개발 및 유지보수· WebSocket을 활용한 실시간 주문 처리 시스템 개발· 매장별 주문 현황을 실시간 모니터링 할 수 있는 통합 관리 시스템 구축

CCTV 기반 ONVIF 통신 체계 구축

미래씨엔씨2020.11 ~ 2023.07
프로젝트 설명 : 스마트팜 IoT 통합 관리 솔루션
프로젝트 배경 : 하이크비전 CCTV 수급 문제로 다양한 제조사의 CCTV를 연동할 필요성이 생겨서 진행하게 됨
· 축사의 CCTV, 환기 시스템, 천장 개폐 등 다양한 IoT 기기를 통합 관리하는 스마트팜 솔루션 개발· Onvif 프로토콜을 활용하여 하이크비전, 다후아, 한화 등 다양한 제조사의 CCTV를 연동하는 CCTV 통합 시스템 구축· 코드리뷰 프로세스 도입을 통해 코드 품질 개선 및 유지보수성 향상· 고객층 연령대를 고려한 UI/UX 개선
